Рукодство по использованию

Для добавления новых нетерминалов в диаграмму необходимо нажать правой кнопкой на рабочем пространстве и выбрать пункт "Add Non-Terminal"



Далее работа осуществляется путем нажатия кнопок управления на самой диаграмме - иконки с плюсами.

Нажатие иконки с плюсом, расположенной на линии, позволяет добавлять новые элементы на эту линию - если название элемента совпадает с названием существующего нетерминала, то стиль добавляемого элемента изменится - прямоугольник будет без скругленных краев

Нажатие иконки с плюсом, расположенной ниже нетерминала, позволяет расширять диаграмму по вертикали. После нажатия на эту кнопку пользователю предлагается выбор начальной и конечной точки расположения новой линии на диаграмме

Выбор осуществляется нажатием на большие круги с черными краями. После выбора точек начала и конца диаграмма расширится по вертикали - высота линии подбирается автоматически таким образом, чтобы линии не пересекали друг друга

На вновь созданную линию так же можно добавлять новые элементы путем нажатия на соответствующую иконку с плюсом - линия увеличится в длине, если места под добавляемый элемент не хватает

Элементы управления можно выключить, нажав правой кнопкой на рабочей и области и выбрав пункт "Toggle Controls", и включить обратно, нажав на "Toggle Controls" еще раз

Приложение поддерживает сохранение (кнопка Save As) и открытие (кнопка Open) файлов в собственном формате (используется xml контейнер), а так же экспорт диаграмм (кнопка Export) в JPG, PNG, SVG, BMP

Диаграммы можно сохранять в local storage - для этого необходимо нажать Save State, а для открытия диаграммы из local storage (если диаграмма там сохранена) необходимо нажать кнопку Retrieve State - возможно, придется обновить страницу, чтобы кнопка появилась - при отсутствии сохраненной диаграммы в local storage кнопка не появляется

Элементы в диаграмме можно двигать и удалять, а так же добавлять различные нестандартные элементы из тулбара слева, однако я не могу гарантировать корректной работы автоматических систем приложения, если пользователь сильно меняет структуру существующей созданной приложением диаграммы.

Запуск редактора на localhost

Здесь приведен способ запуска приложения на локальной машине с помощью node.js:

  • установить node.js
  • загрузить и распаковать куда-либо файлы редактора
  • с активным интернет-соединением запустить localhost.bat в распакованной папке
  • предыдущий шаг достаточно выполнить один раз, после этого запускать приложение можно без интернет-соединения, запуская localhost.bat
